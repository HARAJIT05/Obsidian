# 11. Develop a python command to display the 1st rows of the dataframe using Pandas.

To display the first few rows of a Pandas DataFrame, you use the `.head()` method.

## Command:

The primary command is `df.head()`, where `df` is the name of your DataFrame.

- By default, `.head()` displays the **first 5 rows**.
- You can specify the number of rows you want to see by passing an integer as an argument, like `df.head(n)`, where `n` is the number of rows.

## Example:

Here is a complete Python script that creates a DataFrame and then displays its first rows.

```python
import pandas as pd

# Create a sample DataFrame
data = {
    'Product': ['Apples', 'Oranges', 'Bananas', 'Grapes', 'Cherries', 'Pears', 'Mangos'],
    'Price': [1.2, 0.8, 0.5, 2.5, 3.0, 1.5, 2.0],
    'Quantity': [100, 150, 200, 80, 60, 120, 90]
}
df = pd.DataFrame(data)

print("--- The entire DataFrame ---")
print(df)
print("\n" + "="*30 + "\n")


# --- Python Command to Display the First Rows ---

# 1. Display the default first 5 rows
print("--- Displaying the first 5 rows (default) ---")
first_five_rows = df.head()
print(first_five_rows)
print("\n")

# 2. Display a specific number of first rows (e.g., the first 3 rows)
print("--- Displaying the first 3 rows ---")
first_three_rows = df.head(3)
print(first_three_rows)

```

### Related Command: `.tail()`
Similarly, you can use the `.tail()` method to view the **last** `n` rows of a DataFrame (e.g., `df.tail()` for the last 5 rows).

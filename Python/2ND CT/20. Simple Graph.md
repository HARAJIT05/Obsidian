# 20. Develop a python program to plot a simple graph using Matplotlib.

To plot a simple graph in Python, you can use the **Matplotlib** library, which is the most popular plotting library in the Python ecosystem. The `pyplot` module from Matplotlib provides a simple interface for creating plots.

## Program to Plot a Simple Line Graph

This program will plot a simple line graph showing the relationship between `x` and `y` values, where `y = x^2`.

### Steps:
1.  **Import `matplotlib.pyplot`**.
2.  **Prepare data** for the x and y axes.
3.  **Create the plot** using `plt.plot()`.
4.  **(Optional but Recommended)** Add a title and labels for the axes.
5.  **Display the plot** using `plt.show()`.

### Python Code:

```python
# 1. Import the library
import matplotlib.pyplot as plt

# 2. Prepare the data
# X values (e.g., from -10 to 10)
x_values = list(range(-10, 11))
# Y values (y = x^2)
y_values = [x**2 for x in x_values]

print(f"X values: {x_values}")
print(f"Y values: {y_values}")

# 3. Create the plot
plt.plot(x_values, y_values)

# 4. Add title and labels for clarity
plt.title("Simple Line Graph of y = x^2")
plt.xlabel("X Axis")
plt.ylabel("Y Axis (x^2)")

# You can also add a grid
plt.grid(True)

# 5. Display the plot
# This will open a window showing the graph.
plt.show()
```

### Expected Graph Output:
When you run this script, a window will pop up displaying the graph. The graph will show:
- The **x-axis** labeled "X Axis", ranging from -10 to 10.
- The **y-axis** labeled "Y Axis (x^2)", ranging from 0 to 100.
- A **blue line** forming a parabola, which is the plot of `y = x^2`.
- A **title** at the top of the graph.
- A **grid** in the background to make it easier to read values.

This is a fundamental example, but Matplotlib is capable of creating a vast range of complex and customized visualizations, including scatter plots, bar charts, histograms, and more.

### âœ… **12. Difference between NumPy & Pandas**

| Feature        | NumPy                          | Pandas                                |
| -------------- | ------------------------------ | ------------------------------------- |
| Data Type      | Mostly numerical               | Numeric, text, dates, mixed           |
| Main Object    | ndarray                        | Series & DataFrame                    |
| Focus          | Numerical computing            | Data analysis & manipulation          |
| Missing Values | Limited support                | Strong support (`isna()`, `fillna()`) |
| Performance    | Faster for math operations     | Better for structured tables          |
| Best For       | Scientific & matrix operations | Real-world tabular datasets           |

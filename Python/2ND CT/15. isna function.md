# 15. Explain the isna() function in Pandas with Example.

The `.isna()` function in Pandas is used to **detect missing values**. It returns a boolean same-sized object as the one it's called on, indicating whether each individual value is a "missing" value or not.

"Missing" values in Pandas are typically represented by:
- `np.nan` (Not a Number) for numeric types.
- `None` for object types.
- `pd.NaT` (Not a Time) for datetime types.

`.isna()` is an alias for `.isnull()`. They are the exact same function.

## Key Features:

- **Returns a Boolean Mask:** The output is a DataFrame or Series of `True`/`False` values. `True` means the value at that position is missing, and `False` means it is not.
- **Works on Series and DataFrames:** You can call it on a single column (a Series) or on the entire DataFrame.

## Example:

Let's create a DataFrame with some missing values and use `.isna()` to find them.

```python
import pandas as pd
import numpy as np

data = {
    'Name': ['Alice', 'Bob', 'Charlie', None, 'Eve'],
    'Age': [25, np.nan, 35, 22, 40],
    'Score': [88.5, 92.0, 78.5, np.nan, np.nan]
}
df = pd.DataFrame(data)

print("--- Original DataFrame ---")
print(df)
print("\n" + "="*30 + "\n")

# --- Using .isna() ---

# 1. Use .isna() on the entire DataFrame
print("--- .isna() on the entire DataFrame ---")
missing_values_mask = df.isna()
print(missing_values_mask)
print("\n")

# 2. Use .isna() on a single column (Series)
print("--- .isna() on the 'Age' column ---")
age_missing = df['Age'].isna()
print(age_missing)
print("\n")

# 3. A common use case: Counting missing values
print("--- Counting missing values in each column ---")
missing_counts = df.isna().sum()
print(missing_counts)
print("\n")

# 4. Another common use case: Filtering rows with missing values
print("--- Rows where 'Age' is missing ---")
rows_with_missing_age = df[df['Age'].isna()]
print(rows_with_missing_age)
```

As shown, `.isna()` is a fundamental tool for the first step of handling missing data: **detection**. Once you've identified the missing values, you can then decide whether to drop them (`.dropna()`) or fill them (`.fillna()`).

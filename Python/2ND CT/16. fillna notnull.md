# 16. Explain the fillna() and notnull() function for Pandas.

In Pandas, `.fillna()` and `.notnull()` are two essential functions for handling missing data. `.fillna()` is used to replace missing values, while `.notnull()` is used to detect non-missing values.

---
## `fillna()`

The `.fillna()` function is used to **fill NA/NaN values** with a specified value or method. It is the primary tool for imputing missing data.

### Key Parameters:
- `value`: The value to use to fill holes (e.g., a scalar like 0, or a dictionary to specify different values for different columns).
- `method`: The method to use for filling.
    - `ffill` (forward-fill): Propagate the last valid observation forward.
    - `bfill` (backward-fill): Use the next valid observation to fill the gap.
- `inplace`: If `True`, it modifies the DataFrame in place and returns `None`. If `False` (default), it returns a new, modified DataFrame.

### Example:
```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    'A': [1, 2, np.nan, 4, np.nan],
    'B': [5, np.nan, np.nan, 8, 10],
    'C': ['x', 'y', 'z', np.nan, 'w']
})
print("--- Original DataFrame ---")
print(df)

# 1. Fill with a constant value (e.g., 0)
df_filled_zero = df.fillna(0)
print("\n--- Filled with 0 ---")
print(df_filled_zero)

# 2. Fill with the mean of the column
# Note: This only works on numeric columns
df_filled_mean = df.copy()
df_filled_mean['A'] = df_filled_mean['A'].fillna(df_filled_mean['A'].mean())
print("\n--- Column 'A' filled with its mean ---")
print(df_filled_mean)

# 3. Forward-fill
df_ffill = df.fillna(method='ffill')
print("\n--- Forward-filled ---")
print(df_ffill)
```

---
## `notnull()`

The `.notnull()` function is the direct opposite of `.isna()` (or `.isnull()`). It returns a boolean same-sized object indicating whether the values are **not missing**.

- It returns `True` for any value that is not `NaN`, `None`, or `NaT`.
- It returns `False` for any value that is missing.

### Example:
```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    'A': [1, 2, np.nan],
    'B': [5, np.nan, 8]
})
print("\n--- Original DataFrame for notnull() ---")
print(df)

# 1. Use .notnull() on the DataFrame
notnull_mask = df.notnull()
print("\n--- .notnull() mask ---")
print(notnull_mask)

# 2. Common use case: Select rows that have no missing values in a specific column
complete_rows_A = df[df['A'].notnull()]
print("\n--- Rows where 'A' is not null ---")
print(complete_rows_A)
```

## Summary:

| Function | Purpose | Returns |
| : | : | : |
| **`fillna()`** | To **replace** missing values (`NaN`). | A DataFrame/Series with missing values filled. |
| **`notnull()`** | To **detect** non-missing values. | A boolean DataFrame/Series (`True` if not missing). |
| **`isna()`** | To **detect** missing values. | A boolean DataFrame/Series (`True` if missing). |
